A `retro8-ui` e uma biblioteca de UI retro 8-bit authorada com Tailwind CSS e distribuida como CSS compilado, runtime JS opcional e classes semanticas.

Nesta documentacao, a library continua desacoplada de qualquer framework, enquanto o site de docs ganha um catalog completo de components, navegacao agrupada, pages reais por component e uma base melhor para crescer com guides, changelog e showcases.
