A `retro8-ui` e uma biblioteca de UI retro 8-bit authorada com Tailwind CSS e distribuida como CSS compilado com classes semanticas.

Nesta documentacao, a library continua desacoplada de qualquer framework, enquanto o site de docs ganha navegacao mais fluida, paginas reais por componente e uma base melhor para crescer com guias, changelog e showcases.
