`retro8-ui` is a retro 8-bit UI library authored with Tailwind CSS and shipped as compiled CSS, an optional JS runtime and semantic classes.

In this documentation, the library stays framework-agnostic while the docs site gains a full component catalog, grouped navigation, real component routes and a stronger base for guides, changelogs and showcases.
