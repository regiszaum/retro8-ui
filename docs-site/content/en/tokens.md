---
title: Tokens
description: Colors, spacing, borders, shadows and typography that hold the retro8-ui visual system together.
kicker: Foundation
---

The tokens live in CSS Custom Properties, which keeps components visually consistent and makes the system easier to evolve.

This layer centralizes:

- the main palette
- spacing
- borders and hard shadows
- typography
- interaction states

Because the library is CSS-first, this foundation also makes future theme variations easier without introducing framework wrappers.
